1. Выбрать тему (машинное обучение/нейросети/глубокое обучение) в области (промышленности, медицины и т.д.) и данные

Тема анализа данных - Предиктивная аналитика

Данные - Таблица числовых данных (признаки процесса)
Спрогнозировать данные (бустинг, бэггинг, регрессия)
Визуалиализировать результат

Тема анализа данных - Текстовый анализ

Данные - Набор текстовых данных (неструктурированный, либо колонки текстовые из реляционной БД)
Классификация - по темам
TFIDF - составление словаря по тематике - Вопросно-ответная система, генерация и аннотирование текста
BERT предсказание следующего значения - Вопросно-ответная система, генерация и аннотирование текста
Определение словаря по теме с определением этой темы в дальнейших документах

Тема анализа данных - Анализ изображений

Данные - Набор изображений
Задача - классификация (к каком классу относится изображение), определение объекта, распознавание текста

3. Этапы работы

1. Загрузить данные в приложение (после сбора данных)

2. Предварительная обработка данных под задачу (очистка, нормализация)

3. Обработка перед загрузкой в модель (векторный, числовой вид), дозагрузка доп элементов (например, предобученная BERT)
Делим на тренировочные, тестовые, валидационные данные
Регрессия - обрезать и выделить данные для проверки прогноза

4. Машинное обучение или нейросетейвой алгоритм (глубокое обучение или нейросеть) для тренировочных данных
- Настройка алгоритма
- * Настройка гиперпараметров алгоритма

5. Оценка алгоритма с помощью критериев
Регрессия - MSE, R^2, RMSE, сравнение с прогнозом
Классификация - A, P, R, F1, confusion matrix (P, R), ROC-AUC кривая
Нейросеть - критерии классификации (accuracy), loss, val_loss, P, R, F1, confusion matrix (P, R)

6. Выгрузка модели. Визуализировать как работает и проверять на новых данных

7. Сделать вывод по работе модели
